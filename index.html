Petster Exotic

Social networking web app for those who own exotic pets and are looking for a pet sitter, or those looking to find exotic pet-sitting opportunities. Users can register an account as a pet sitter or owner, update account information, save favorite profiles, search by location or description, contact other users by email or video call, edit a calendar and use Paypal to tip pet sitters.

  • Front End:
    • React & Material UI to create a dynamic, user-friendly interface
    • Includes packages such as sweetalert, axios, moment & more!
  • Back End:
    • Node, Express and Mongoose to connect with a MongoDB database, seeded with data from faker
    • JWT and Passport for authentication and authorization, and bcrypt for password hashing.

Winner of the Wyncode's Pitch Day contest. Visit the deployed site here!

Meme Park

Integrates the GIPHY API to allow a user to search, browse and share memes on social media.

Consulted with a UX/UI partner on a wireframe and site design. Front end coded with React and Bootstrap. Back end uses a Node & Express server to fetch data from the GIPHY API.

Tested endpoints with Postman and deployed to Heroku via the GitHub auto-connect option so that we could immediately know which coding changes affected the deployment build.

{ About }

Hi! I'm Liz - a full stack developer and Wyncode alum. Before making a change into the tech industry, I was working in retail as a department manager and in human resources. I've always loved programming and had taught myself how to build small websites with HTML/CSS/Javascript, but I really wanted to take that to the next level and start a full-fledged career in development. Now, after graduating Wyncode, I'm building MERN stack web applications and am 100% ready to kickstart that career!

My most recent big project was Petster Exotic, a social networking web app for exotic pet owners and sitters to easily connect and communicate. My idea for the project came from my own struggles owning three adorable snakes and experiencing how difficult it is to find a sitter and leave care instructions. I built this project as part of a team of three for Wyncode's Pitch Day Contest, where we were challenged to create a web application using the MERN tech stack in 2 weeks. The project won the contest, after we presented the site in front of an audience 120+ people and answered questions about our code from a panel of judges.
